Від цього курсу очікую отримати достатній рівень знання мов програмування JavaScript/TypeScript для легкого вивчення популярних фреймворків(React, Angular, Vue.js). Освоїти основні концепції front-end розробки, такі як DOM (Document Object Model), інтерактивність та асинхронність. Вдосконалити вміння використовувати інструменти для розробки: редактори коду, системи контролю версій. Навчитись працювати з інструментами тестування.
